Table/Control Dataclass Prototype

Before vs After

BEFORE
# Table: 100% generic tbl.attributes["rowCnt"] # str tbl.children[0] # GenericElement (row?) tbl.children[0].children[0] # GenericElement (cell??) # Control: 1 typed field ctrl.control_type # str ctrl.attributes["id"] # str
→
AFTER
# Table: typed structure tbl.row_cnt # int tbl.rows[0] # TableRow tbl.rows[0].cells[0].col_addr # int tbl.rows[0].cells[0].width # int # Control: typed fields ctrl.control_type # str ctrl.id # Optional[str] ctrl.name # Optional[str]
