Normalization Flow

# HWPX 파일 열기 흐름 HWPX ZIP → header.xml (bytes) ↓ normalize_hwpml_namespaces(data) # byte-level replace │ ├─ 2016 URI → 2011 URI (기존, 7개 매핑) ├─ 2024 URI → 2011 URI (신규, 7개 매핑) │ ↓ parse_xml(normalized_data) ↓ 기존 코드 (2011 URI 기준) → 정상 동작
