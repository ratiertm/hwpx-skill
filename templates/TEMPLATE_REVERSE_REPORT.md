## 템플릿 생성 규칙 요약

### 1. 페이지 설정
```python
# 정부 서식 표준
page = {"width": 59528, "height": 84188, "landscape": "WIDELY"}
margin = {"left": 5104, "right": 5104, "top": 4252, "bottom": 4252,
          "header": 2836, "footer": 2836}
```

### 2. 표 크기 계산
```python
text_area = page["width"] - margin["left"] - margin["right"]
table_width = text_area - 280  # outMargin 140×2
```

### 3. 서식 유형별 그리드 전략

| 유형 | 컬럼 수 | 전략 |
|------|---------|------|
| 단순 양식 (의견제출서) | 4 | 균등 분할 |
| 정부 서식 (신청서) | 9~14 | 불균등, 구분선 컬럼 포함 |
| 복잡 양식 (근로지원인) | 10 | 불균등, 부분 역산 필요 |

### 4. 셀 역할 매핑
```python
# 정부 서식 구조
TITLE:  row=0, colSpan=전체, charPr height 크게
LABEL:  ①~⑨ 포함, borderFill 두꺼운 테두리
INPUT:  빈 텍스트, borderFill 얇은 테두리
STATIC: "즉시", "없음" 등 고정값
```
